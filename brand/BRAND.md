## Logo Usage Rules

### Clear Space
Maintain a clear space equal to half the height of the logo icon.

### Minimum Size
- Digital: 100px height minimum
- Print: 25mm height minimum

### Do's and Don'ts

| ✅ Do | ❌ Don't |
| :--- | :--- |
| Use the provided SVG files | Stretch or distort the logo |
| Maintain clear space | Change the colors |
| Use on contrasting backgrounds | Add shadows or effects |
| Scale proportionally | Use outdated versions |
